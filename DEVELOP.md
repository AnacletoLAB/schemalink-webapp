# neo4j arrows monorepo - managed with nx.dev

✨ **This workspace has been generated by [Nx, a Smart, fast and extensible build
system.](https://nx.dev)** ✨

## Development server

1. Create `apps/arrows-ts/.env` and populate using `apps/arrows-ts/.env.sample`
   as an example;
2. Run `nx serve arrows-ts` for a dev server;
3. Navigate to <http://localhost:4200/>.

The app will automatically reload if you change any of the source files.

## Understand this workspace

Run `nx graph` to see a diagram of the dependencies of the projects.

## Remote caching

Run `npx nx connect-to-nx-cloud` to enable [remote caching](https://nx.app) and
make CI faster.
